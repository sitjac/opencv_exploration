# OpenCV Exploration

A modern C++ projects to explore the OpenCV library and its features.

- Use **CMake** as the build system
- Use **Conan** as the package manager
- Use **GoogleTest** for unit testing
- Use **Doxygen** for documentation generation
- Use **GitHub Actions** for CI/CD


## Getting Started

These instructions will get you a copy of the project up and running on your local
machine for development and testing purposes.

### Prerequisites

* **CMake v3.28.3** - found at [https://cmake.org/](https://cmake.org/)

* **Conan v1.65.0** - found at [https://conan.io/](https://conan.io/)

* **C++ Compiler** - found in your system, such as *GCC* or *Clang*.s

> ***Note:*** C++ Compiler needs to support at least the **C++17** standard

### Build

```bash
cmake -B build
cmake --build
```
